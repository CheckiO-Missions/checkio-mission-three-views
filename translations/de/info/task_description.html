Das Volumen eines Objekts wird fast aus drei Ansichten abgeleitet.

Sie erhalten drei Ansichten (Vorderseite, rechte Seite, oben) als Eingabewerte. Jede dieser Ansichten ist eine Zeichenfolge, die aus A bis Y besteht. Diese Buchstaben sind in einem 5x5-Gitter in zeilenweiser Reihenfolge angeordnet.

Sie müssen das Gesamtvolumen des Objekts zurückgeben.

HINWEIS:
  • Die Antwort ist der maximal mögliche Wert.

Beispiele: 

assert three_views('BCDGHILMNQRSVWX', 'CHMRW', 'LMN') == 15
assert three_views('GHILMNQRS', 'GHILMNQRS', 'GHILMNQRS', ) == 27

Eingabe: Die drei Ansichten (Vorderseite, rechte Seite, oben) (3 Strings)

Ausgabe: Das Gesamtvolumen der Objekte (Eine ganze Zahl)
