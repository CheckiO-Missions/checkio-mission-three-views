Le volume d'un objet est presque dérivé de trois vues.

Vous disposez de trois vues (avant, côté droit, haut) comme valeurs d'entrée. Chacune d'entre elles est une chaîne de caractères composée de A à Y. Ces lettres sont classées par ordre croissant dans une grille de 5 x 5.

Vous devez indiquer le volume total de l'objet.

REMARQUE:
  • La réponse est la valeur maximale possible.

Exemples: 

assert three_views('BCDGHILMNQRSVWX', 'CHMRW', 'LMN') == 15
assert three_views('GHILMNQRS', 'GHILMNQRS', 'GHILMNQRS', ) == 27

Entrée: Les trois vues (avant, côté droit, haut) (3 chaînes)

Sortie: Le volume total des objets (un entier)
